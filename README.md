British Airways Data Science Simulation (Forage)

Overview

This project is my completed work for the British Airways Data Science Job Simulation offered by Forage. The goal of the simulation is to explore customer review data, extract insights, and provide data-driven recommendations that can support service improvement at British Airways.
The repository contains my full workflow including data cleaning, analysis, visualizations, and insights.

Objectives

Understand customer sentiment and overall satisfaction.

Identify common themes in positive and negative reviews.

Build visual summaries that highlight key patterns.

Recommend actions that can help improve the customer experience.

Dataset

The dataset includes British Airways customer reviews with:

Ratings

Review titles and text

Flight details

Reviewer attributes

All processing and analysis steps are included in the notebook.

Approach
1. Data Preparation

Cleaned missing and inconsistent values

Prepared text fields for analysis

Normalized rating and sentiment indicators

2. Exploratory Data Analysis

Rating distribution

Most common words and themes

Comparison of positive vs negative reviews

3. Text and Sentiment Insights

Word frequency analysis

Identification of key pain points

Extraction of high-impact service areas

4. Visualizations

Bar charts and trend plots

Word clouds

Highlighted patterns across review segments

5. Recommendations

Based on insights from the data:

Improve check-in and boarding experience

Enhance in-flight service consistency

Address recurring issues around delays and baggage

Focus on communication around service updates
